# Interior Design Service Website - Design Recreation

A recreation of a modern interior design service website built with Next.js. This project demonstrates the implementation of a professional interior design layout with clean and elegant user interface elements.

## 🎯 Project Overview

This is a recreation of a modern interior design website layout, built to practice and demonstrate Next.js development skills. The project focuses on implementing responsive design, component architecture, and modern web development practices.

## 🚀 Features Implemented

- **Design Recreation**: Faithful recreation of the original design layout
- **Responsive Implementation**: Mobile-first approach with all device compatibility
- **Component Architecture**: Modular React components following best practices
- **Modern Styling**: CSS implementation of the original design elements
- **Interactive Elements**: Hover effects and smooth transitions
- **Clean Code Structure**: Well-organized and maintainable codebase
- **SEO Optimized**: Built with Next.js for optimal search engine performance

## 🛠️ Technologies Used

- **Next.js** - React framework for production
- **React** - JavaScript library for building user interfaces
- **CSS3** - Modern styling and animations
- **JavaScript (ES6+)** - Modern JavaScript features
- **Responsive Design** - Mobile-first approach

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/interior-design-website.git
cd interior-design-website
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 🔧 Customization

### Updating Content

1. **Hero Section**: Edit the main headline and description in `components/Hero.js`
2. **Services**: Update service offerings in `components/Services.js`
3. **Portfolio**: Add new project images and descriptions in `components/Portfolio.js`
4. **Contact Information**: Update contact details in `components/Contact.js`

### Styling

- Global styles are in `styles/globals.css`
- Component-specific styles are in `styles/components/`
- Color variables can be updated in the CSS root variables

### Images

- Add new images to the `public/images/` directory
- Update image paths in components as needed
- Optimize images for web performance
